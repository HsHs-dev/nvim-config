# Hassan's Neovim Setup

## The IDE Killer

---

# Why Neovim?

- Blazing fast startup
- Fully keyboard driven
- Infinitely customizable
- Runs anywhere

---

# Languages Supported

- C / C++ via clangd
- Java via nvim-jdtls
- TypeScript / JavaScript
- Python via pyright
- Assembly, Bash, Lua

---

# Key Features

- LSP completions
- DAP debugging (F5 to start)
- Telescope fuzzy finder
- Flash jump navigation
- Git via Lazygit

---

# Keymaps

| Key          | Action     |
| ------------ | ---------- |
| `s`          | Flash jump |
| `<leader>fg` | Live grep  |
| `<leader>gg` | Lazygit    |
| `<F5>`       | Debug      |
